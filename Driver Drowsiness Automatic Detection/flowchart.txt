----------------------------
FATIGUE SCORE CALCULATION
----------------------------
  -- Calculate Fatigue Score (0-100)
       -- EAR Component (0-60 points)
           - EAR >= 0.25: 0 points (Fully alert)
           - EAR 0.22-0.25: 0-20 points (Slight drowsiness)
           - EAR 0.18-0.22: 20-45 points (Moderate drowsiness)
           - EAR 0.12-0.18: 45-60 points (Severe drowsiness)
           - EAR < 0.12: 60 points (Very closed eyes)

       -- MAR Component (0-35 points)
           - MAR <= 0.5: 0 points (No yawning)
           - MAR 0.5-0.65: 0-12 points (Slight yawning)
           - MAR 0.65-0.9: 12-30 points (Yawning)
           - MAR > 0.9: 30-35 points (Deep yawning)

       -- Eye Closure Component (0-25 points)
           - eye_closed_frames * 0.7 (max 25 points)

       -- Total Score
           - fatigue_score = EAR_score + MAR_score + Eye_Closure_score
           - Penalty: +15 points if eyes closed + mouth open

FRAME PROCESSING PIPELINE
----------------------------
  -- Processing Loop
       1. Capture frame from webcam
       2. Resize frame
       3. Convert to grayscale
       4. Detect face(s) using dlib
       5. Extract facial landmarks
       6. Compute EAR and MAR
       7. Compute fatigue score
       8. Apply alarm/notification logic
       9. Draw overlays and status labels
       10. Store processed frame
       11. Update detection_data dictionary
       12. Sleep briefly for steady processing
